## 3 Advanced Prompts Ready to Copy and Paste

Below are real, advanced examples you can use directly in **AI2You** (or other AI platforms) to create high-impact content.

### Prompt for Reels (short viral video)

```md
Act as a viral strategist for Instagram Reels. Generate a complete 
30-second script for a Reel about “3 ways to style a basic jacket in 
winter”, including:

- An irresistible hook in the first 3 seconds
- Suggested visual transitions (zoom, quick cuts, on-screen text)
- Text overlay for each of the 3 tips
- A strong final CTA encouraging comments and shares
- A suggestion for a currently trending audio
- 8–10 hashtags optimized for the sustainable fashion niche
```
**Why does this prompt work so well?**
It delivers a complete narrative structure combined with technical elements that maximize retention and virality within the Reels algorithm.

## Prompt for Stories (launch / engagement)

```md
Create a sequence of 3 Instagram Stories to launch a free e-book about 
“AI-Powered Digital Marketing for Content Creators.”

Structure:

1. Story 1: Teaser with an interactive question
(suggestion: poll or question sticker) + eye-catching emoji
2. Story 2: Key benefits highlighted with short text overlay
suggested thematic image
3. Story 3: Direct CTA with download link
urgency trigger (“available today only!”)

Use a motivational tone, strategic emojis, and sticker suggestions to increase interactivity.
```
**Why is this prompt so powerful?**
It transforms passive views into concrete actions (responses, clicks), making it ideal for launches and lead generation.
